# Bean & Brew Cafe Menu

A simple, responsive menu page for a coffee shop, built with HTML and CSS.

## Features
- Clean two‑column layout (desktop) that stacks on mobile.
- Categorized menu (Hot Drinks, Pastries & Desserts).
- Accessible image with `alt` text.
- Subtle hover effects on menu items.
- Responsive design with a mobile‑first approach.

## Files
- `index.html` – the main HTML document.
- `style.css` – all styling.

## How to View
Simply open `index.html` in any modern web browser. No server required.

## Future Enhancements (Ideas)
- Add prices to menu items.
- Include icons for dietary preferences (vegan, gluten‑free).
- Implement a dark mode toggle.
- Use JavaScript to filter menu items.
